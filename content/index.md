This is the documentation home. Use the sidebar to navigate.

## What's included

- **MDX rendering** — write content as `.md` or `.mdx` in `/content`
- **Auto-generated sidebar** from your folder structure
- **Full-text search** powered by Pagefind (built at `npm run build`)
- **Dark mode** with system preference detection
- **Syntax highlighting** with `rehype-pretty-code` + Shiki
- **Copy button** on every code block

## Quick start

```bash title="Terminal"
npm install
npm run dev
```

Then drop a markdown file into `content/` and it'll show up in the sidebar.
